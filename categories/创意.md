## 1. 图片生成结构化提示词（Midjourney / DALL-E）

**Prompt：**
```
[主体描述], [场景/环境], [光线/色调], [构图/角度], [风格参考], [额外参数]

Example:
A photorealistic 3D collector figure of a man in flannel shirt and jeans, displayed on a wooden desk beside a large monitor, dramatic studio lighting, ultra-detailed fabric texture, Sideshow Collectibles quality, --ar 4:5 --v 7
```
> 来源：Medium — Complete List of Prompts & Styles for MidJourney 2026 Edition
> 用法：图片生成提示词的固定配方：主体 → 场景 → 光线 → 构图 → 风格 → 参数。顺序影响权重，越靠前的词权重越高。--ar 控制宽高比，--v 指定模型版本。

## 3. Midjourney 风格化艺术肖像

**Prompt：**
```
Create an artistic portrait with these specifications:

Subject: [人物描述]
Art style: [油画/水彩/赛博朋克/水墨/3D渲染]
Lighting: [伦勃朗光/蝴蝶光/侧光]
Color palette: [色调]
Vibe: [氛围]
--ar [宽高比] --v [版本] --s [风格化程度]

Parameters to adjust:
- --stylize (s): how artistic vs literal
- --chaos (c): variation in composition
- --weird (w): experimental output
- --iw: image weight (if using reference image)
```
> 来源：Medium — Complete MidJourney Prompt List 2026
> 用法：Midjourney 艺术肖像生成，含风格化参数调优。

## 8. 虚构作品世界观构建
> 📅 2026-07-06

**Prompt：**
```
Build a fictional world for my story:

Genre: [奇幻/科幻/dystopia/历史架空]
Core premise: [核心设定]
Tone: [黑暗/光明/灰色]

Generate:
1. World rules — what physics/magic/society laws govern this world

5. Technology/magic system — how it works, its limits, and cost
6. Map description — key geographic features and their significance
7. Language/culture quirks — unique idioms, customs, taboos
```

## 11. 配色方案生成
> 📅 2026-07-06

**Prompt：**
```
Generate a color palette for:

Project type: [品牌/UI/插画/室内设计]
Base color: [主色]
Mood: [温暖/专业/活力/优雅]
Accessibility: [WCAG AA compliant]

Provide:
1. Primary palette (5 colors with hex codes)
2. Secondary palette (3 accent colors)
3. Neutral palette (5 shades of gray)
4. Semantic colors (success, warning, error, info)
5. Usage recommendations (which color for what)
6. Dark mode variant
7. Gradient combinations (3 options)

For each color, include: hex, RGB, HSL values, and WCAG contrast ratio against white/black.
```
> 来源：AI Prompt Library — Color Palette Generator (2026)
> 用法：配色方案生成，含无障碍对比度检查。

## 12. 动画关键帧描述
> 📅 2026-07-06

**Prompt：**
```
Describe keyframes for an animation:

Subject: [动画主体]
Action: [动作描述]
Duration: [X秒]
Style: [2D/Motion Graphics/3D/Stop Motion]
Mood: [氛围]

For each keyframe, provide:
1. Timecode (seconds or frames)
2. Subject position and rotation
3. Camera angle and movement
4. Lighting changes
5. Timing/Easing function
6. Sound/sync cue

Include start pose, anticipation, action, follow-through, and end pose.
```
> 来源：AI Prompt Library — Animation Keyframes (2026)
> 用法：动画关键帧描述，适合生成 AI 动画或向动画师传达创意。
