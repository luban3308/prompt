## 1. 房地产交易初步筛选

**Prompt：**
```
Screen this property deal for initial viability:

Property type: [住宅/商业/多户/土地]
Asking price: [要价]
Location: [地址/区域]
Square footage: [面积]
Year built: [建造年份]
Current occupancy: [当前入住率]
Listed rent: [挂牌租金]
Estimated expenses: [预估支出]
Financing terms: [贷款条件]

Perform:
1. Quick cap rate calculation (NOI / price)
2. Cash-on-cash return estimate
3. 1% rule test (rent vs price ratio)
4. Market comparable check (price/sqft vs comps)
5. Red flag identification (deferred maintenance, rent control, environmental)
6. Go/No-Go recommendation with rationale
7. Critical questions for further due diligence
```
> 来源：SurePrompts — AI Prompts for Real Estate 2026
> 用法：快速筛选房产交易，在深入尽职调查前判断是否值得跟进。

## 2. 五年房产投资预测模型

**Prompt：**
```
Build a 5-year investment projection for:

Purchase price: [购买价]
Down payment: [首付]
Loan terms: [利率/期限/类型]
Rental income: [当前及预期租金]
Operating expenses: [管理费/物业税/保险/维修]
Annual appreciation: [年增值率预期]
Vacancy rate: [空置率假设]
Exit strategy: [5年后出售/再融资/持有]

Generate:
1. Year-by-year cash flow projection (income - expenses - debt service)
2. Equity build-up schedule (principal paydown + appreciation)
3. IRR (Internal Rate of Return) calculation
4. Cash-on-cash return for each year
5. Sensitivity analysis (appreciation ±2%, vacancy ±5%)
6. Tax implications summary (depreciation, capital gains)
7. Breakeven occupancy rate
8. Comparison to alternative investments (S&P 500, bonds)
```
> 来源：SurePrompts — Real Estate Financial Modeling 2026
> 用法：五年投资预测模型，含现金流、IRR、敏感性分析和税务考量。

## 3. 房产尽职调查清单

**Prompt：**
```
Generate a comprehensive due diligence checklist for:

Property: [房产地址/类型]
Purchase price: [购买价]
Transaction type: [直接购买/1031交换/ foreclosure]
Timeline: [尽调截止日期]
Location: [城市/州]

Categories to cover:
1. Physical inspection (structural, roof, HVAC, plumbing, electrical, pest)
2. Financial verification (rent roll, expense statements, tax returns)
3. Legal review (title, zoning, permits, litigation, HOA docs)
4. Environmental assessment (Phase I, soil, asbestos, lead paint)
5. Market analysis (comps, absorption rate, demographic trends)
6. Insurance review (coverage adequacy, claims history)
7. Property management assessment (current mgr performance, contracts)
8. Code compliance (ADA, fire safety, building codes)

For each item, include: what to look for, who can provide it, and a pass/fail threshold.
```
> 来源：SurePrompts — Real Estate Due Diligence 2026
> 用法：生成适合特定物业类型的完整尽职调查清单，确保不遗漏关键项。

## 4. 装修投资回报分析

**Prompt：**
```
Analyze the ROI of a renovation project for:

Property: [房产描述]
Current condition: [现状描述]
Renovation scope: [装修范围]
Cost estimate: [装修预算]
Timeline: [施工周期]
Financing: [自有资金/贷款]
Exit strategy: [出租增值/出售]

For each renovation item, calculate:
1. Cost (materials + labor)
2. Expected value increase (dollar amount and %)
3. Payback period (months)
4. ROI ranking (highest return items first)
5. Risk factors (permits, contractor reliability, material delays)

Also provide:
- Overall project ROI estimate
- Renovation vs. market comparison (what buyers/renters in this area expect)
- Minimum list price / rent needed to break even
- Contingency budget recommendation (10/15/20%)
- Before/after value estimate
```
> 来源：SurePrompts — Real Estate Renovation Analysis 2026
> 用法：装修项目ROI分析，识别最高回报的改造项目，避免过度装修。
