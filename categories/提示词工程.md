## 1. 六标准提示词质量评分（Prompt Rubric）

**Prompt：**
```
Score this prompt against the 6-Criteria Rubric before running it:

Prompt: [待评估的提示词]

1. CLARITY: Is the task unambiguous? (1-5)
2. SPECIFICITY: Are constraints and format defined? (1-5)
3. CONTEXT: Is enough background provided? (1-5)
4. ACTIONABILITY: Can the AI act on it immediately? (1-5)
5. MEASURABILITY: Would you know if the output is good? (1-5)
6. COMPLETENESS: Does it include all necessary elements? (1-5)

Total: /30. For scores below 4 on any criterion, suggest specific improvements.
```
> 📅 2026-07-11
> 来源：KeepMyPrompts — Score Any Prompt in 3 Minutes (2026)
> 用法：在运行提示词之前预检质量，而不是等输出结果不好了再责怪提示词。

## 3. 提示词工程最佳实践清单

**Prompt：**
```
Check this prompt against the 2026 Prompt Engineering Best Practices checklist:

Prompt: [待检查的提示词]

1. ☐ Success criteria stated upfront?
2. ☐ Structured input with clear delimiters?
3. ☐ 1-3 examples provided (if applicable)?
4. ☐ Exact output format specified?
5. ☐ Role/identity defined?
6. ☐ Constraints and boundaries named?
7. ☐ Negative instructions (what NOT to do)?
8. ☐ Evaluation criteria included?

If any box is unchecked, suggest how to add it.
```
> 📅 2026-07-11
> 来源：PromptBuilder — Prompt Engineering Best Practices (2026)
> 用法：八项清单快速检查，确保不遗漏关键要素。
