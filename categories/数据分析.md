## 11. A/B 测试结果解读
> 📅 2026-07-06

**Prompt：**
```
Interpret my A/B test results:

Test name: [测试名称]
Metrics tracked: [指标列表]
Control variant data: [对照组数据]
Test variant data: [实验组数据]
Sample size: [样本量]
Test duration: [测试时长]

5. Novelty effect assessment (did the effect decay over time?)
6. Recommendation (ship/kill/iterate/run longer)
7. Power analysis (was the sample size sufficient?)
```
> 来源：SurePrompts — Experimentation (2026)
> 用法：A/B 测试结果的专业解读。

## 13. 回归分析解释
> 📅 2026-07-06

**Prompt：**
```
Interpret this regression analysis output:

Model type: [线性回归/逻辑回归/多元回归]
Coefficients: [粘贴回归系数表]
R-squared: [R²值]
P-values: [p值]
Sample size: [样本量]
Variables: [自变量描述]

Provide:
1. Model fit assessment (how well does it explain the data?)
2. Key drivers identification (which variables matter most?)
3. Effect size interpretation (practical vs statistical significance)
4. Multicollinearity check (VIF values)
5. Residual analysis (patterns, outliers, heteroskedasticity)
6. Model limitations (omitted variable bias, endogeneity)
7. Actionable insights from the analysis
```
> 来源：SurePrompts — Statistical Modeling (2026)
> 用法：回归分析结果解读。
