## 20. A/B 测试统计解读

**Prompt：**
```
Help me interpret A/B test results.

Test description: [测试描述]
Metric: [核心指标]
Control group: [对照组样本量 + 转化率/均值]
Variant group: [实验组样本量 + 转化率/均值]
Statistical significance threshold: [α值 — 如0.05]
MDE (Minimum Detectable Effect): [最小可检测效应量]
Duration: [测试持续时间]
Early stopping: [是否提前停止]

Provide:

1. Significance test — is the result statistically significant? (p-value)
2. Effect size — how big is the difference practically?
3. Confidence interval — range of the true effect
4. Bayesian interpretation — probability that variant is better (if applicable)
5. Practical significance — is the effect large enough to matter for business?
6. Common pitfalls check — multiple testing, peeking, sample ratio mismatch
7. Segmentation analysis — does the effect vary by segment?
8. Recommendation — launch, iterate, or run a follow-up test?

If the result is not significant: calculate required sample size for a conclusive test.
```
> 📅 2026-07-12
> 来源：A/B Testing / AI Data Science (2026)
> 用法：A/B测试结果解读——不只p值，还看效应量、置信区间、实用显著性和常见陷阱。
