## 1. 健身/运动计划生成

**Prompt：**
```
Create a workout plan for:

Goal: [增肌/减脂/力量/耐力]
Experience: [新手/中级/高级]
Days per week: [X天]
Equipment: [健身房/居家/徒手]
Time per session: [X分钟]
Injuries/limitations: [受伤或限制]

Provide:
1. Weekly split (which muscle groups on which days)
2. Exercises for each session with sets/reps
3. Progression plan (how to increase over time)
4. Warm-up and cool-down
5. Nutrition notes
6. Recovery recommendations
```
> 来源：AI Academy — Personal Life Prompts 2026
> 用法：个性化健身计划，含渐进超负荷设计。

## 4. 第一封约会消息

**Prompt：**
```
Write a first message for a dating app:

Their profile says: [粘贴对方个人简介]
Shared interests: [共同兴趣]
My goal: [破冰/幽默/真诚展示]

- Reference something specific from their profile
- Be conversational, not generic

```

## 15. 正念冥想引导脚本
> 📅 2026-07-06

**Prompt：**
```
Write a guided meditation script:

Duration: [X分钟]
Focus: [呼吸/身体扫描/慈心/专注]
Setting: [晨间/睡前/压力时刻]
Experience level: [初学者/有经验者]
Background: [用户当前状态]

Structure:
1. Grounding (settling in, 30 seconds)
2. Breath awareness (anchor to breath, 1 minute)
3. Main practice (guided visualization or body scan)
4. Acknowledgement (noticing thoughts without judgment)
5. Closing (gentle return to room, 30 seconds)

Tone: calm, warm, permissive. Use natural pauses (marked as [pause]).
```
> 来源：AI Academy — Wellness Prompts (2026)
> 用法：正念冥想引导脚本，从扎根到结束的完整引导。
