## 1. YouTube 视频创意与内容策略

**Prompt：**
```
Act as my YouTube strategist. Based on what you know about [我的频道/领域], my audience, and my business:

1. Analyze my niche — what topics are underserved?
2. Generate 10 video ideas that fill content gaps in my niche.
3. For each idea: video title (optimized for clicks AND search), viewer intent, competition level (low/medium/high), and which of my existing videos it could link to.
4. Flag which 3 ideas to prioritize — I want to batch-film the "search" videos first.
```

> 用法：先找内容缺口再生成创意，优先制作能长期带来搜索流量的视频。

## 2. YouTube 爆款标题与钩子生成

**Prompt：**
```
Generate a hook library for my [领域] YouTube channel.

Analyze my best-performing videos: [列出3-5个最受欢迎视频的标题/主题].

Build 7 distinct hook formats drawn from what works in my niche:
1. Each with a one-line description of when to use it
2. An example of how I could open my next video with each format
3. The emotional trigger each hook activates (curiosity, surprise, controversy, aspiration, fear)

Also provide 10 click-worthy title options for my upcoming video about [主题].
```
> 📅 2026-07-11
> 来源：Forbes — 5 ChatGPT Prompts To Create Viral YouTube Scripts (2026)
> 用法：从你自己的成功视频中提取钩子模式，比通用模板更契合你的频道风格。

## 3. YouTube 脚本撰写（保持创作者声音）

**Prompt：**
```
Write a full YouTube script for a video about [主题].

Target length: [X分钟]
My channel style: [描述你的风格]

Structure:
1. Hook (first 15 seconds — must grab attention immediately)
2. Introduction (what we're covering and why it matters)
3. Main body (3-5 sections with pattern interrupts every 60-90 seconds)
4. Conclusion (key takeaways + call to action)
5. End screen suggestion

Rules:
- Write in my voice, not generic AI voice
- Include visual cues in [brackets]
- Mark "pattern interrupts" where I should change scene/shot
- End every section with a mini-hook for the next section
```
> 📅 2026-07-11
> 来源：Techpresso / Storyflow — YouTube Script Writing Prompts (2026)
> 用法：含模式中断标记的专业YouTube脚本模板，保持观众留存率。

## 4. YouTube SEO 与描述优化

**Prompt：**
```
Optimize my YouTube video metadata for searchability.

Video topic: [主题]
Target keywords: [关键词列表]

Generate:
1. Video title (max 60 chars, keyword-forward, clickable)
2. Description (first 2 lines must hook and include primary keyword)
3. Tags (mix of broad and specific, max 15)
4. Timestamp chapters with titles
5. End screen and cards recommendations
6. One thumbnail concept (describe text overlay and visual)

Use this title format: [keyword] + [benefit/curiosity] + [brand/reason to click]
```

> 来源：Forbes / Techpresso — YouTube SEO Prompts (2026)
> 用法：完整的YouTube SEO元数据生成，标题/描述/标签/时间戳一应俱全。

## 5. YouTube 增长策略与内容日历

**Prompt：**
```
Create a 30-day content growth plan for my YouTube channel.

Channel niche: [领域]
Current subscribers: [数量]
Average views: [数量]
Upload frequency: [频率]
Current strength: [擅长什么]
Current weakness: [需要改进什么]

Plan should include:
1. Weekly theme for 4 weeks
2. 2 video concepts per week (one "searchable" evergreen + one "trending")
3. Thumbnail strategy per video
4. Community engagement plan (comments, community posts)
5. Collaboration/Shout-out strategy
6. Success metrics to track per week
```

## 6. 短视频平台矩阵策略

**Prompt：**
```
Help me create a cross-platform short video strategy.

YouTube Shorts: [当前频道/状态]
TikTok: [当前账号/状态]
Instagram Reels: [当前账号/状态]
Content niche: [内容领域]
Available content: [已有素材/内容库]
Production capacity: [每周可产出视频数]
Target audience: [各平台的目标受众]

Design a platform strategy:

1. Platform-specific content optimization — format, duration, aspect ratio, caption style per platform
2. Content repurposing — how to adapt one idea across all 3 platforms
3. Platform-native trends — trending audio, effects, formats per platform
4. Cross-promotion — how to drive audience between platforms
5. Upload schedule — optimal posting times per platform
6. Hashtag strategy — per-platform hashtag tiers (broad, niche, branded)
7. Analytics — which metrics matter per platform (not just views)
8. Platform-specific hooks — what works on TikTok vs Reels vs Shorts

Create a content calendar template for one week of multi-platform content.
```

## 7. 缩略图 A/B 测试方案

**Prompt：**
```
Design a thumbnail A/B testing framework for my video content.

Channel type: [YouTube/TikTok/Instagram]
Content niche: [内容领域]
Current thumbnail style: [当前缩略图风格]
Tools available: [Canva/Photoshop/AI生成]
Weekly video output: [每周视频数量]

Create the thumbnail testing framework:

1. Test variables — which elements to test (face expression, text overlay, color scheme, composition)
2. Hypothesis formulation — what do we think will improve CTR and why
3. Design variants — how to create meaningful variants (not trivial differences)
4. Sample size — how many impressions needed for statistical significance
5. Testing duration — how long to run the test (avoid early stopping)
6. Success metrics — click-through rate, retention rate, watch time
7. Documentation template — record each test and result
8. Iteration — how to build on winning patterns

Also generate 3 thumbnail variant concepts for a sample video topic with descriptions.
```
> 📅 2026-07-12
> 来源：YouTube Optimization / AI Video Strategy (2026)
> 用法：缩略图A/B测试框架——系统化测试变量、统计显著性要求和迭代优化流程。

## 8. 直播脚本与互动设计

**Prompt：**
```
Design an interactive livestream script for my broadcast.

Platform: [Twitch/YouTube Live/TikTok Live]
Stream type: [游戏/教学/聊天/产品发布]
Stream duration: [时长]
Target audience: [受众描述]
Stream goals: [关注增长/销售/社区互动]
Frequency: [首次/定期]

Create the script:

1. Pre-stream checklist — tech check, lighting, audio, backup plan
2. Opening segment — how to welcome latecomers, set expectations
3. Content segments — timed blocks with transitions between them
4. Interaction points — planned moments to read chat, ask questions, polls
5. Engagement mechanics — giveaways, shoutouts, challenges, Q&A slots
6. Call to Action moments — subscribe, follow, share, buy (naturally integrated)
7. Moderation plan — chat rules, mod assignments, handling trolls
8. Closing segment — wrap-up, next stream tease, end screen

Include a "conversation starter" bank — 10 questions to ask the chat when engagement dips.
```

> 来源：Live Streaming / AI Video Prompts (2026)
> 用法：直播脚本设计——从设备检查到互动设计，含聊天冷场时的话题急救包。

## 9. YouTube 社区运营指南

**Prompt：**
```
Help me build and manage a YouTube community around my channel.

Channel topic: [频道主题]
Subscriber count: [订阅数]
Current engagement: [评论量/点赞率]
Community tab: [是否使用]
Discord/Other: [有其他社区平台吗]
Upload schedule: [发布频率]

Create a community management plan:

1. Comment strategy — how and when to reply, what comments to pin
2. Community posts — what to post between videos (polls, updates, teasers)
3. Super Chat/Membership — if monetized, how to structure perks
4. Feedback loops — how to get input on what subscribers want to see
5. User spotlight — featuring community creations or comments
6. Negative feedback handling — criticism vs trolling, response protocol
7. Community guidelines — rules that set the tone without being restrictive
8. Growth milestones — celebrate milestones with the community

Draft 5 community posts for the next 2 weeks that drive conversation.
```

> 来源：Community Building / AI Video (2026)
> 用法：YouTube社区运营全攻略——评论管理、社区动态、粉丝互动，把观众变成社区成员。
